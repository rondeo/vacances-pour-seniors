## PHP_CodeSniffer <a name = "php_codesniffer"></a>

Include as dependency **PHP_CodeSniffer** in your `composer.json` file:

```json
{
	"require-dev": {
		"squizlabs/php_codesniffer": "*"
    }
}
```

Then type `composer update`.
